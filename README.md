# solana-web3

useful script to learn and test solana web3

0. understandable
1. run independent
2. copy to reuse


## how to use
1. To install dependencies:

```bash
bun install
```

2. Set Env
```bash
cp env .env
```


```
# 私钥，用于发送交易
PRIVATE_KEY=
# 另一个钱包公钥，接收交易
PUBLIC_KEY=92U3aM8kRqihutqFWNDGxmWXxMHDrnAyFEMYXcKmvKcD
# 开发或测试网rpc
RPC=https://api.devnet.solana.com
```

3. To run:

```bash
bun run ***.ts
```
