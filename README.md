# solana-web3

solana web3 script for copy anytime
可复制 solana web3脚本

1. account manage & balance  -  帐号管理和余额查询
2. token info & balance - token信息和余额查询
3. transaction - 交易
   
## how to use
1. To install dependencies:

```bash
bun install
```

2. Set Env
```bash
cp env .env
```

3. To run:

```bash
bun run ***.ts
```
